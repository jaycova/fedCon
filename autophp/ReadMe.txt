============ Instructions ============

1. Create a database (for example, codexworld) and import the "SQL/skills.sql" file in this database.

2. Open the "search.php" file ===> Specify the database host ($dbHost), username ($dbUsername), password ($dbPassword), and name ($dbName) as per your MySQL server credentials.

3. Open the "index.php" file on the browser ===> Start typing in the input field ===> Autocomplete suggestions will appear under the input field ===> Select a value from auto-suggestion and submit.
